# laravel-rect-install


  ## composer require laravel/ui
  ## php artisan ui react
  ## npm install
  ## npm run dev
  
# Add webpack.mix.js

```
mix.react('resources/js/app.js', 'public/js')
  .sass('resources/sass/app.scss', 'public/css');
  
  ```
# Add IN HTML
  ```
 <script src="{{ asset('js/app.js') }}" defer></script>
 ```
